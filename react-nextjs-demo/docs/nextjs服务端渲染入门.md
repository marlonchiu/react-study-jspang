## 框架特点

* 使用后端渲染
* 自动进行代码分割`（code splitting）`，以获得更快的网页加载速度
* 简洁的前端路由实现
* 使用webpack进行构建，支持模块热更新`（Hot Module Replacement）`
* 可与主流Node服务器进行对接`（如express）`
* 可自定义babel和webpack的配置
